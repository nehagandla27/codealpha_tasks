# Cloud-Based Bus Pass System

## Project Overview
Cloud-Based Bus Pass System developed for CodeAlpha Cloud Computing Internship.

## Features
- User Registration
- User Login
- Bus Pass Booking
- Route Selection
- Fare Calculation
- SQLite Database Storage

## Technologies Used
- Node.js
- Express.js
- SQLite3
- EJS
- CSS

## Installation

npm install

node app.js

Open:

http://localhost:3000
